# proyecto 1

Este proyecto actualmente esta ocupando SASS como lenguaje de estilos.

## Requerimientos

- nodejs
- npm
- sass

### Como correrlo
en una terminal, ubicarse en el directorio del proyecto y correr el siguiente comando
```
sass --watch sass/main.scss index.css
```
